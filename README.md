# Boonrodchoo House - Construction Services
เว็บไซต์อย่างเป็นทางการของบริษัท Boonrodchoo House และระบบ E-Service สำหรับบริหารจัดการงานรับเหมาก่อสร้าง

## Features (ฟีเจอร์หลัก)
1. **Corporate Website:** แนะนำบริการและผลงานของบริษัท
2. **Construction E-Service:** ระบบคำนวณและสรุปยอดเงินรับเหมาก่อสร้าง (รองรับการออกไฟล์ PDF/PNG/Excel)
3. **Worker E-Service:** ระบบคำนวณค่าแรงคนงานและการเบิกล่วงหน้า (รองรับการทำสลิปเงินเดือน)

## Tech Stack
* HTML5 / CSS3
* React.js (Standalone via CDN)
* Tailwind CSS
* html2canvas & jsPDF (สำหรับออกเอกสาร)
